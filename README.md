## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 稳定的网络连接

### 安装步骤
1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html`
3. 选择你喜欢的AI女友开始聊天

### 使用说明
1. **选择AI女友**：在左侧边栏点击任意AI女友卡片
2. **开始对话**：在底部输入框输入消息，点击发送或按Enter键
3. **切换女友**：随时可以切换到其他AI女友，每个都有独立的聊天记录
4. **个性化设置**：点击设置按钮调整主题和字体大小

## 🛠️ 自定义配置

### 修改API密钥
在 `script.js` 文件中的 `apiKeys` 对象修改相应的密钥：

```javascript
this.apiKeys = {
    deepseek: 'your-deepseek-api-key',
    kimi: 'your-kimi-api-key',
    zhipu: 'your-zhipu-api-key',
    step: 'your-step-api-key',
    tencent: 'your-tencent-api-key',
    tongyi: 'your-tongyi-api-key',
    doubao: 'your-doubao-api-key'
};
```

### 添加新的AI女友
1. 在 `girlfriends` 对象中添加新的AI配置
2. 在HTML中添加对应的卡片元素
3. 在 `callAI` 方法中添加API调用逻辑
4. 实现对应的API调用函数

### 自定义主题
在 `style.css` 中修改CSS变量或添加新的主题类：

```css
body.theme-custom {
    --primary-color: #your-color;
    --gradient-primary: linear-gradient(...);
}
```

## 🐛 常见问题

### Q: AI回复很慢或失败
A: 检查网络连接，确认API密钥有效，部分模型可能有访问限制。

### Q: 聊天记录丢失
A: 聊天记录存储在浏览器本地，清除浏览器数据会导致记录丢失。

### Q: 移动端体验不佳
A: 建议使用Chrome或Safari浏览器，确保JavaScript已启用。
